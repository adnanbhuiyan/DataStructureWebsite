Linked List

A linked list is a linear data structure where elements are stored in nodes connected by pointers. Unlike arrays, linked lists allow dynamic memory allocation and efficient insertion and deletion operations.

Why Use Linked Lists?

Linked lists are particularly useful for:

  • 1. Dynamic memory allocation.
  • 2. Efficient insertion and deletion operations.
  • 3. Implementing stacks, queues, and other abstract data types.
  • 4. Managing data where memory allocation is unpredictable.

Code Examples

  • 1. // How to create a linked list node 
    class Node {
    constructor(data) {
        this.data = data;
        this.next = null;
    }
}
  • 2. // How to insert at the beginning 
    let head = null;
const newNode = new Node(10);
newNode.next = head;
head = newNode;
  • 3. // How to traverse a linked list 
    let current = head;
while (current !== null) {
    console.log(current.data);
    current = current.next;
}
  • 4. // How to delete a node 
    function deleteNode(key) {
    let current = head;
    let prev = null;

    while (current !== null && current.data !== key) {
        prev = current;
        current = current.next;
    }

    if (current === null) return; // Key not found

    prev.next = current.next;
}
