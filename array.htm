Arrays

An array is a data structure that contains a group of elements. These elements are stored at contiguous memory locations and can be accessed using indices. Arrays are widely used because of their efficiency in accessing and manipulating data.

Why Use Arrays?

Arrays are crucial in programming for the following reasons:

  • 1. Easy access to elements using indices.
  • 2. Efficient use of memory for fixed-size data.
  • 3. Supports a wide range of algorithms, including searching and sorting.
  • 4. Foundational structure for more complex data structures like matrices and tables.

Code Examples And Give it a Try below

  • 1. // How to create an array 
    const array = [1, 2, 3, 4, 5];
  • 2. // How to add an element to an array 
    array.push(6); // Adds 6 to the end
  • 3. // How to remove an element from an array 
    array.pop(); // Removes the last element
  • 4. // How to access an element in an array 
    const firstElement = array[0]; // Accesses the first element
  • 5. // How to find the length of an array 
    const length = array.length; // Returns the number of elements
  • 6. // How to iterate over an array 
    array.forEach(element => {
    console.log(element);
});
  • 7. // How to remove an element by index 
    array.splice(2, 1); // Removes 1 element at index 2
  • 8. // How to check if an element exists in an array 
    const exists = array.includes(3); // Returns true if 3 exists
